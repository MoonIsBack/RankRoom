# 📌 START — Inhaltsverzeichnis

## Wofür ist das?

Das hier ist dein Lernheft für RankRoom. Es ist **keine Entwicklerdoku für Fremde**,
sondern eine Gedächtnisstütze für dich selbst.

Wenn du in drei Monaten wieder reinschaust und nicht mehr weißt, was `useTierLists.js`
eigentlich macht — dann schlägst du hier nach.

## Wie du es benutzt

Jede Datei in diesem Ordner ist **eine OneNote-Seite**. Datei öffnen, alles markieren,
in OneNote einfügen. Fertig.

Die Nummern vorne sind die Reihenfolge. Du musst sie aber nicht der Reihe nach lesen —
spring dahin, wo dein Problem gerade ist.
